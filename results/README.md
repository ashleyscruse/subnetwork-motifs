# Results

Generated outputs from experiments. Large files (models, predictions) are gitignored.

## Expected contents after running the pipeline

- `predicted_pi.csv` -- GAT-predicted pi values per gene family
- `baseline_results.csv` -- AUC-ROC for all baselines
- `ablation_results.csv` -- ablation study results
- `motif_catalog.csv` -- the significant motif catalog
- `figures/` -- generated plots for the paper
